Student Entry Registration Form

Description summarizing the functionalities

This student registration system allows users to input their details including name, class, address, and contact number into a form. Upon submitting the form, the entered data is displayed in a table below the form. Users can edit or delete each entry in the table. Clicking "Edit" populates the respective row data back into the form for editing, while clicking "Delete" removes the entire row from the table.
